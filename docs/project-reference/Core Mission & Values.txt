Core Mission & Values
Primary Mission
Empower users to complete a personal comprehensive health check / cyber remediation.
This is a personal Cyber Security remediation app, disguised as user friendly fun experience. This app is designed to help people secure themselved from cyber threats. Users get value from this by securing their digital footprint. Learning comes secondary, and more focused in Level 3 of the app.
Core Values & Principles
Action-First Approach
Learning will come secondary, only when absolutely required.
Accessibility Focus
App is intended for non-technical audiences to be able to understand and complete level 1-2 checks. Level 3 is for advanced users.
Complete Independence
App is completely vendor & company agnostic and therefore can provide unbiased recommendations. Will not unnecessarily sell you something.
Expert Foundation
The knowledge and small team behind the app comes from accredited IT professionals working in the cyber security industry.
Comprehensive Support
Other offerings will include blog posts (i.e. what is a VPN and do you need it), playbooks (what to do if your email is hacked) etc.
Design Concepts & Implementation Framework
Progressive Disclosure & User Experience
    • Progress Tracking - Visual progress bars, completed modules marked 
    • Interactive Elements - Step-by-step actions users can complete immediately 
    • Platform-Specific Guidance - Automatic detection of user's device/OS for tailored instructions 
    • Contextual Action Design - Integrate with device settings where possible. Instead of just telling users to "enable auto-updates," provide direct links or in-app guidance to the specific settings 
    • Task-Oriented Information Architecture - Organize by user goals rather than technical categories. "Protect my accounts" is more intuitive than "Authentication systems" 
    • Practice Scenarios - Safe simulated phishing emails, fake scam calls to test knowledge 
Engagement & Motivation
    • Progress Celebrations - Positive reinforcement at regular intervals 
    • Achievement System - Badges for completing modules, streak counters 
    • Refresher Reminders - Gentle notifications for periodic security updates (i.e. monthly security posture assessments) 
Professional Interface Design
    • Clean, Scannable Layout - Use plenty of white space, clear typography hierarchies, and consistent spacing. Security information can feel overwhelming, so clean design is crucial 
    • Professional Visual Language - Use modern, clean iconography and avoid overly cute/playful elements that might undermine trust in professional contexts 
Quick Reference Guide for Decision Making
When developing features, ask:
    • ✅ Does this help users complete their cybersecurity health check? 
    • ✅ Is this simple enough for non-technical users? 
    • ✅ Are we staying vendor-neutral? 
    • ✅ In the main health check (core app function), are we prioritizing action over education? 
    • ✅ Is our expert knowledge evident but not overwhelming? 
Red flags to avoid:
    • ❌ Teaching concepts without immediate actionable steps 
    • ❌ Recommending specific paid products unnecessarily 
    • ❌ Using technical jargon without clear explanations 
    • ❌ Creating educational content that doesn't lead to security improvements, unless in the separate Insights area
