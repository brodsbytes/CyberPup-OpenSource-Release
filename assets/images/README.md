# Images Directory

This directory contains images used throughout the CyberPup app.

## CyberPup Mascot

**File:** `cyberpup-mascot.png`

**Description:** A futuristic CyberPup mascot with dark blue body and light blue glowing accents.

**Requirements:**
- Size: Approximately 200x200 pixels (or larger, will be scaled down)
- Format: PNG with transparency support
- Style: Futuristic, robotic dog with glowing elements

**Usage:** The mascot appears on the InitialAuditScreen welcome step with a glowing background effect.

## Adding the Image

1. Save your CyberPup mascot image as `cyberpup-mascot.png`
2. Place it in this directory: `assets/images/cyberpup-mascot.png`
3. The app will automatically load and display it with the proper styling

## Styling Features

The mascot is displayed with:
- 200x200 pixel container with rounded corners
- Light blue glow effect around the image
- Centered horizontally on the screen
- Top margin for proper spacing
- Subtle background glow to make it stand out against the dark theme
